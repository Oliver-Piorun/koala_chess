# Koala chess
A 2D chess game written in Rust.

## Supported platforms
- Windows (OpenGL ES 3.2)

## Technical goals
- Use as few dependencies as possible
- Keep an eye on performance

## Personal goals
- Learn more about Rust, rendering and game development
